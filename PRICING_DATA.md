# Pricing Data Mappings

## Supported AI Tools
The frontend maps the following tools to the backend for auditing:

1. **ChatGPT (OpenAI)**
   - Plans: Plus ($20/mo), Team ($30/seat/mo), Enterprise
   - Vendor URL: chatgpt.com/pricing
2. **Claude (Anthropic)**
   - Plans: Pro ($20/mo), Team ($30/seat/mo)
   - Vendor URL: anthropic.com/pricing
3. **GitHub Copilot**
   - Plans: Individual ($10/mo), Business ($19/seat/mo)
   - Vendor URL: github.com/pricing

## Frontend Display Reference
The frontend references these internal IDs (`toolId`) to render tool-specific icons and human-readable names across the UI.
