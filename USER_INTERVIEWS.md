# User UX Feedback

## Interview 1: Engineering Manager
**Confusion Points:** The user was initially confused by having to input the exact monthly spend per tool. 
**Resolution:** We added intelligent defaults on the frontend so selecting "ChatGPT Team" auto-fills $30/seat.

## Interview 2: Operations Lead
**Dashboard Clarity:** Loved the immediate "Total Annual Savings" metric in the top left. Found the chart visualizer helpful for understanding the breakdown.
**Recommendations Usefulness:** Feedback was that recommendations needed to be more actionable. We updated the UI to highlight "Downgrade" vs "Consolidate" actions with distinct badge colors.
